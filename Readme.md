Neste TDE você deverá criar um programa que leia um arquivo de forma binária; ou seja, que leia byte a byte o conteúdo dele. Eu sugiro que você edite um arquivo com o Notepad e coloque vários caracteres acentuados e que tenha várias linhas. Se quiser, veja a tabela UTF-8 (https://www.utf8-chartable.de/unicode-utf8-table.pl) e coloque alguns caracteres diferentes. Você deverá abrir o arquivo para leitura, ler os bytes presentes no arquivo e colocar o código numeríco de cada byte.
